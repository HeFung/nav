Node框架

Npm

JavaScript 的包管理和分发工具。

Pnpm

快速的、节省磁盘空间的包管理工具。

Yarn

Yarn 是一个软件包管理器，还可以作为项目管理工具。

Node.js

Node.js是一个基于Chrome V8引擎的JavaScript运行环境。

Express.js

Express.js是Node.js最久远的后端框架之一，主要用于创建Web应用程序和RESTful API。

Koa.js

Koa.js是一个由Express.js原班人马打造的更小、更强大、更富有表现力的web框架。

NestJS

NestJS是一个用于构建高效、可扩展的Node.js服务器端应用程序的框架。

Midway

Midway是阿里巴巴-淘宝前端架构团队，基于渐进式理念研发的Node.js框架。

Egg

为企业级框架和应用而生。

Hapi

Hapi是一个用于构建RESTful API的Node.js框架。

Fastify

Fastify高度专注于以最少的开销和强大的插件架构提供最佳的开发人员体验。

AdonisJS

AdonisJS是一个用于构建高效、可扩展的Node.js应用程序的全栈框架，特别适合开发Web应用和API。
