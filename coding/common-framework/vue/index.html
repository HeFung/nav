Nuxt官网

Vite是Vue3官方推荐的前端构建工具，以其快速的热更新和构建速度而闻名，极大提升了开发体验。

pnpm

在包管理器的选择上，首选 PNPM，相比于 NPM，它能解决很多问题，例如幽灵依赖、安装速度快、节省空间、天然支持Monorepo。

Vue Router V4

Vue Router是Vue.js的官方路由管理器，它允许你构建单页面应用（SPA）。

Vue Router V3

Vue Router V4是官方的路由管理器，专为Vue3设计。它提供了一种声明式的方式来定义导航和页面路由，使得构建单页面应用（SPA）变得更加容易。

Vue Devtools

一个旨在增强Vue开发人员体验的工具，它是一个功能强大的通用工具，可以显着提高您在使用Vue应用程序时的生产力和调试能力。

一个以Markdown为中心的静态网站生成器。

Vitest

vue-loader是webpack的一个loader，允许你用单文件组件(sfc)的格式创建Vue组件并打包。

Axios

易用、简洁且高效的http库。Axios是一个基于promise的网络请求库，可以用于浏览器和node.js。

Alova

一个流程简化的下一代请求工具，它可以将你的 API 集成工作流从 7 个步骤极致地简化为 1 个步骤，你只需要选择 API 即可使用。
